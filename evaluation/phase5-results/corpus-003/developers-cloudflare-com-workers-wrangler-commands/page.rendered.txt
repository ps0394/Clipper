Wrangler
 offers a number of commands to manage your Cloudflare Workers.

The core Wrangler commands for creating, developing, and deploying Workers are on the 
Workers commands page
. This includes 
wrangler dev
, 
wrangler deploy
, 
wrangler versions
, and more.

How to run Wrangler commands

wrangler <COMMAND> <SUBCOMMAND> [PARAMETERS] [OPTIONS]

Since Cloudflare recommends 
installing Wrangler locally
 in your project (rather than globally), the way to run Wrangler will depend on your specific setup and package manager.

 npm 
 yarn 
 pnpm
