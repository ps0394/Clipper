## Code Examples:

### Add Semantic HTML Structure:
```html
<!-- Before (problematic) -->
<div class="content">
  <h1>Page Title</h1>
  <p>Content here...</p>
</div>

<!-- After (semantic) -->
<main>
  <article>
    <h1>Page Title</h1>
    <p>Content here...</p>
  </article>
</main>
```

### Fix Heading Hierarchy:
```html
<!-- Before (hierarchy jumps) -->
<h1>Main Title</h1>
<h3>Subsection</h3>  <!-- ❌ Skip H2 -->
<h2>Section</h2>     <!-- ❌ Wrong order -->

<!-- After (proper hierarchy) -->
<h1>Main Title</h1>
<h2>Section</h2>
<h3>Subsection</h3>  <!-- ✅ Logical flow -->
```
